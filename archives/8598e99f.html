# 038 result

n. 结果；后果；vi.（因...）发生；导致

# 相关词

# consequence

# aftermath

# fallout

n. 后果；余波

# 039 court

n. 法院；宫廷；法庭；全体出庭人员；（网球等的）球场；v. 渴望

# courtship

n. 求爱；（动物的）求偶

# 相关词

# defense/-ce

n. 起诉；原告方

# prosecutor

n. 检察官；公诉人

# judge

n. 法官；裁判员；鉴定人；v. 判断；审判；评价

# judg(e)ment

n. 判断力；评价；判决

# trail

n. 审判；试验；考验；v. 试验；试用

# appeal

n. 上诉；吸引力；呼吁；vi. 有吸引力；上诉；呼吁

# plead

v. 恳求；辩护；解释

# 040 market

# 041 place

n. 地方；地表的某处；（尤指占用或者空着的）座位；（速度比赛或竞赛获胜者的）名次；vt.（小心或有意）放置；使（人）处于某位置；以某种态度对待（或看待）；下赌注；或名次

# placement

n.(对物件的) 安置；（对人的）安排

# replace

vi. 代替；（用...）替换

# replacement

n.（尤指工作中的）代替者；替代品；替换

# misplaced

adj. 不合时宜的；给错对象的

# 相关词

# substitute

n. 替代者；v.（以...）代替

# 042 set

vt. 确定；使处于某种状况；创立；放；把故事情节安排在；摆放餐具；n. 一组，一套（配套使用的东西）；电视机；adj. 套（餐）；安排好的

词组：

# setback

n. 挫折

# setter

n. 制订者

# settle

v. 结束（争论、争端等）；定居；（最终）决定；使处于舒适的地位；（使）平静下来

# settlement

n. 定居地；解决；（解决纷争的）协议

# unsettle

vt. 使心神不宁

# 043 service

n. 服务；国家政府部门；vt. 检修（车辆、机器）；为... 提服务

# 同根词

# serve

v. 为（人、机构）服务；上（食品、饮料）；对... 有作用；履行义务；可用作

# deserve

v. 值得

# observe

vt. 观察；注意到；评述；遵守（规则、法律等）

# observation

n. 观察；评论

# observer

n. 观察者

# preserve

vt. 维护；保存；贮存；n. 专门领域

# reserve

vt. 保留；预定（座位，房间等）；拥有；n. 储备（量）；（动植物）保护区

# conservation

n. 保护；节约

# 044 view

vt. 以... 看待；观看 n.（个人的）看法；（理解或思维的）方法；视野；风景

# viewpoint

n. 观点；角度

# interview

n. 面试；采访；vt. 采访；对（某人）进行面试

# interviewer

n. 采访者；主持面试者

# review

n. 评审；评论；回顾；vt. 审查；写（关于书籍、戏剧、电影等的）评论；（尤指为准备考试而）温习

# reviewer

n. 评论家；审查者

# 相关词

# perspective

n. 观点；客观判断力；远景

# landscape

n.（陆上，尤指乡村的）风景；乡村风景画；格局

# prospect

n. 可能性；前途；前景；（竞赛中的）有望获胜者；vi. 勘探
