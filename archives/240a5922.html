🍘若你决定灿烂，山无拦，海无遮！

# 1. 若你决定灿烂，山无拦，海无遮。（撒贝宁）

这句话适合用于 “意志力”、“梦想” 等主题，例如：一个有坚定信念的人无疑是最强大的，若他决定灿烂，山无拦，海无遮，乌云都为他让道。

# 2. 任何一个梦想，都有不被打击，不被嘲笑的权利。（撒贝宁）

这句话适合用于 “梦想”、“勇敢逐梦” 等主题，例如：恰如撒贝宁所言：“任何一个梦想，都有不被打击，不被嘲笑的权利。” 每一个梦想都是伟大的，每一个追梦人都是值得尊敬的，当你播下一粒梦想的种子，就把他人的冷言冷语当作耳旁风吧，你尽管辛勤浇水施肥，直至梦想结出硕果。

# 3. 纵使眼前迷雾蒙蒙，我仍然相信会有美丽的风景。（撒贝宁）

这句话适合用于 “乐观”、“走出迷雾” 等主题，例如：纵使眼前迷雾蒙蒙，我仍然相信会有美丽的风景，只要我策马扬帆勇往直前，总会有云开雾散的一天，那时我便可以将美景尽收眼底。

# 4. 如果命运是世界上最烂的编剧， 你就要做你自己最好的演员。（撒贝宁）

这句话适合用于 “命运”、“活出自我” 等主题，例如：如果命运是世界上最烂的编剧， 你就要做你自己最好的演员，毕竟我命由我不由天，命运的钥匙永远在我们手上，只有我们自己能够决定我们能活得有多精彩。

# 5. 如果你认准了，不撞南墙不回头，撞到了南墙把它撞穿，继续走。（撒贝宁）

这句话适合用于 “执着”、“坚定” 等主题，例如：如果你认准你认定一个目标，你就得不撞南墙不回头，就算是撞到了南墙也得把它撞穿，继续往前走，直到你实现它。

# 6. 父母看似任由我闯荡去按照自己的路走其实他们的目光一直盯着我的背影。（撒贝宁）

这句话适合用于 “父 / 母爱”、“人生” 等主题，例如：父母看似任由我闯荡去按照自己的路走其实他们的目光一直盯着我的背影。我们就是父母放的风筝，他们不断地放长手中的线，任我们在高空自由地飞翔，其实，纵使我们飞得再高再远，他们也永远在地下望着高飞的我们。

# 7. 眼前的挑战，或许险如珠穆朗玛。但多年以后，你会明白这不过是生命中众多起落的丘陵之一。（撒贝宁）

这句话适合用于 “克服困难”、“勇于挑战” 等主题，例如：眼前的挑战对你来说，或许险如珠穆朗玛，但只要你勇往直前，一步步将这所谓的高峰翻越，再往回看时，你会明白这不过是生命中众多起落的丘陵之一。

# 8.“不要输在起跑线” 这句话，我是不会赞同的，因为只有百米赛跑才会在乎起跑。而人生是一场马拉松比赛，只有最先跑到终点的那一位才是人生的赢家。（撒贝宁）

这句话适合用于 “终点和起点”、“人生” 等主题，例如：我永远不会赞同 “不要输在起跑线” 这句话，因为只有百米赛跑才会在乎起跑，而人生是一场马拉松比赛，只有最先跑到终点的那一位才是人生的赢家，赢在起点的人未必能赢在终点，而在起点落后的人也能够拔得头筹。
