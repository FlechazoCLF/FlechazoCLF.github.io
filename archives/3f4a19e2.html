# 小柴带你学 AutoSar 系列一、基础知识篇（7）术语

# AUTOSAR 专业术语

  • AUTOSAR（AUTomotive Open System ARchitecture） - 汽车开放系统架构
  • BSW（Basic Software） - 基础软件
  • MCAL（Microcontroller Abstraction Layer） - 微控制器抽象层
  • ECU（Electronic Control Unit） - 电子控制单元
  • RTE（Run-Time Environment） - 运行时环境
  • SWC（Software Component） - 软件组件
  • VFB（Virtual Functional Bus，虚拟功能总线）是 AUTOSAR 架构中的概念，用于在不依赖具体硬件的情况下，实现软件组件之间的通信和交互，支持系统设计和验证。
  • OSEK（Offene Systeme und deren Schnittstellen für die Elektronik im Kraftfahrzeug） - 开放式系统及其在车辆电子中的接口
  • CDD（Complex Device Driver）：复杂设备驱动，用于实现复杂设备的驱动程序。
  • DEM（Diagnostic Event Manager） - 诊断事件管理器
  • DCM（Diagnostic Communication Manager） - 诊断通信管理器
  • DET（Default Error Tracer）：默认错误追踪器，记录系统中发生的错误信息。
  • COM（Communication） - 通信模块
  • PDU（Protocol Data Unit） - 协议数据单元
  • CAN（Controller Area Network） - 控制器局域网
  • LIN（Local Interconnect Network） - 本地互连网络
  • UDS（Unified Diagnostic Services） - 统一诊断服务
  • ASIL（Automotive Safety Integrity Level） - 汽车安全完整性等级
  • FIBEX（Field Bus Exchange Format） - 现场总线交换格式
  • ARXML（AUTOSAR XML） - AUTOSAR 标准 XML 文件格式
  • NVRAM（Non-Volatile Random-Access Memory） - 非易失性随机存取存储器
  • TP（Transport Protocol） - 传输协议
  • PWM（Pulse Width Modulation） - 脉宽调制

# RH850 专业术语

RH850 - 瑞萨电子开发的一系列微控制器

  • MCU（Microcontroller Unit） - 微控制单元
  • ADC（Analog Digital Converter）
  • CPU（Central Processing Unit） - 中央处理单元
  • ECC（Error Correction Code） - 错误校正码
  • WDT（Watchdog Timer） - 看门狗定时器
  • DMA（Direct Memory Access） - 直接内存访问
  • FPU（Floating Point Unit） - 浮点运算单元
  • ADC（Analog-to-Digital Converter） - 模数转换器
  • DAC（Digital-to-Analog Converter） - 数模转换器
  • CAN-FD（CAN with Flexible Data-Rate） - 具备灵活数据速率的控制器局域网
  • PWM（Pulse Width Modulation） - 脉宽调制
  • SPI（Serial Peripheral Interface） - 串行外设接口
  • I2C（Inter-Integrated Circuit） - 集成电路间通信协议
  • UART（Universal Asynchronous Receiver-Transmitter） - 通用异步收发器
  • PLL（Phase-Locked Loop） - 锁相环
  • GPIO（General-Purpose Input/Output） - 通用输入输出
  • RTC（Real-Time Clock） - 实时时钟
  • SRAM（Static Random-Access Memory） - 静态随机存取存储器
  • NVM（Non-Volatile Memory） - 非易失性存储器
  • EEPROM（Electrically Erasable Programmable Read-Only Memory） - 电可擦除可编程只读存储器

# 其他相关术语

  • AES（Advanced Encryption Standard） - 高级加密标准：一种对称加密算法，用于保护数据的安全性。
  • ARP（Address Resolution Protocol） - 地址解析协议：用于将 IP 地址解析为物理地址（MAC 地址）。
  • CRC（Cyclic Redundancy Check）：循环冗余校验，用于数据校验和错误检测。
  • DES（Data Encryption Standard）：数据加密标准，一种对称加密算法，已被 AES 取代。
  • DoIP（Diagnostics over IP） - 基于 IP 的诊断：通过 IP 网络进行诊断通信的协议。
  • DTC（Diagnostic Trouble Code） - 诊断故障码：用于标识车辆电子系统中的特定问题。
  • E2E（End-to-End） - 端到端：描述完整的通信路径或数据传输过程。
  • FIFO（First In, First Out） - 先进先出：数据结构或方法，最先进入的数据最先被处理。
  • FOTA（Firmware Over-The-Air） - 无线固件更新：通过无线通信技术远程更新设备固件。
  • GPT（General Purpose Timer） - 通用定时器：用于时间测量和定时功能的定时器。
  • HTTP（Hypertext Transfer Protocol） - 超文本传输协议：用于万维网上数据通信的协议。
  • ICMP（Internet Control Message Protocol） - 因特网控制报文协议：用于网络设备之间传递控制消息的协议。
  • ICOM（In-vehicle COMmunication） - 车内通信：指车内电子控制单元（ECU）之间的通信。
  • ICU（Input Capture Unit） - 输入捕获单元：用于捕获输入信号并记录其时间的硬件单元。
  • ISO（International Organization for Standardization） - 国际标准化组织：制定和发布国际标准的机构。
  • ISR（Interrupt Service Routine） - 中断服务程序：处理硬件中断的程序。
  • MPU（Memory Protection Unit） - 存储保护单元：用于保护内存区域，防止非法访问。
  • MTU（Maximum Transmission Unit，最大传输单元）是指网络通信中单个数据包或帧的最大大小，以字节为单位。
  • NM（Network Management） - 网络管理：管理和监控网络节点和通信的模块或功能。
  • OBD（On-Board Diagnostics） - 车载诊断系统：用于监控和报告车辆性能和故障的系统。
  • OEM（Original Equipment Manufacturer） - 原始设备制造商：生产和销售其他公司品牌产品的公司。
  • PCI
    • （Peripheral Component Interconnect） - 外围组件互连：用于连接计算机及其外围设备的总线标准。
    • （Protocol Control Information，协议控制信息）是指在数据通信中用于控制和管理数据传输的元数据。PCI 通常包含在数据帧或数据包的头部，用于携带关于数据的传输、路由、优先级等控制信息。
  • SHM
    • （Shared Memory） - 共享内存：多个进程可以访问的公共内存区域。
    • （System Health Monitoring，系统健康监控）是一种技术和工具，用于实时监控和评估系统的运行状态和健康状况。它主要应用于确保系统的可靠性、性能和安全性，通过检测、记录和分析各种系统参数和事件，及时发现和预防潜在的问题。
  • SM（State Machine） - 状态机：根据当前状态和输入决定下一个状态的系统。
  • SOME/IP（Scalable service-Oriented Middleware over IP） - 基于 IP 的可扩展服务中间件：用于汽车电子系统中基于 IP 的通信中间件。
  • TCP（Transmission Control Protocol） - 传输控制协议：一种面向连接的、可靠的通信协议。
  • TTCAN（Time-Triggered CAN） - 时间触发 CAN：基于时间触发机制的 CAN 通信协议。
  • TTL
    • （Transistor-Transistor Logic） - 晶体管 - 晶体管逻辑：一种数字电路设计中的逻辑门设计方式。
    • （Time To Live，生存时间）是指数据包在网络中可以存活的最大时间或跳数，用于防止数据包在网络中无限循环。
  • UDP（User Datagram Protocol） - 用户数据报协议：一种无连接的、简单的通信协议。
  • UML（Unified Modeling Language） - 统一建模语言：用于软件工程中的标准化建模语言。
  • VSM
    • （Vehicle System Modeling） - 车辆系统建模：对车辆系统进行建模和仿真的过程。
    • （Vehicle State Manager）
  • VMM（Virtual Memory Management） - 虚拟内存管理：操作系统用于管理虚拟内存的功能。
  • XCP（Universal Measurement and Calibration Protocol） - 通用测量和校准协议：用于车辆控制单元（ECU）数据测量和校准的协议。
  • XP（Extreme Programming） - 极限编程：一种敏捷软件开发方法。
