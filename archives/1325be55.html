⛄安卓手机安装可到云+内网穿透，可道云搭建☃

首页 技术教程🎃 ksweb

采用工具 / 软件等如下：
1、 一部旧的安卓机（推荐 root）
Ps：我采用的是小米 note5
2、可到云源码
3、ksweb / 超级主机
4、域名（ps：我用的是 freenom 提供的免费域名）
5、sakera frpc（樱花 frpc）/ngrok（目前可能只能用钉钉了）
Ps：推荐樱花 frpc 速度比 ngrok 快（亲测在 Google play 下载 frp）
6、frpc 文件
7、超级终端（termux）
常用文件下载：链接：https://pan.baidu.com/s/186X4BOU3it4yJdRO38NxRA
提取码：pabw
复制这段内容后打开百度网盘手机 App，操作更方便哦
上述工具准备好就开始我们的搭建之旅吧！
方法及步奏：
方法 1（kwseb+sakura frpc）
1、 安装 ksweb，并启动 lighttpd 服务
配置主机，端口设为 80（未 root 用户这步跳过）

2、 打开手机文件管理器，找到 htdocs 文件夹把之可到云源码解压到此目录

3、 打开路由器管理界面进行 ip 与 mac 绑定，比如我的是 http://tplogin.cn/，(如没有可查看路由器下面，一般都有，如果没有可以用手机查看路由器的默认网关进行登陆）登陆密码没有的可以问父母要


4、 比如我绑定的是 192.168.1.188 且因为用 wordpress 做主站故源码位置为 192.168.188/pan，之前 ksweb 端口设置的是 80，那么浏览器直接输入 ip 地址，如果之前端口没改就输入 192.168.1.188:8080（8080 是一开始的默认端口，可自行修改）

5、 上述操作成功后便可以进行下一步操作，去 freenom 注册你的免费域名







6、 接下来访问樱花 frpc 官网，注册账号，并添加映射操作如下



7、 去刚刚注册域名的网站开启 dns 解析，选择 a 记录，地址为樱花 frpc 提供的主机地址

8、 把文件放置在这个软件的 home 目录（data/data/com.termux/files/home），打开超级终端，输入./frpc -c 123.ini ps: 我的配置文件为 123.ini，

9、 接下来访问你注册的域名看看是否成功（ps:dns 解析需要时间，若无法打开请过几分钟后再试）
方法 2（超级主机 + ngrok）ps: 需 root
1、 安装超级主机
2、 把可到云源码放置在手机根目录的 data/data/xiaoqidun.anmpp/files/root/android.wwwroot/localhost 中
3、 按照方法 1 的 3-5 步
4、 注册 ngrok 账号添加隧道


5、 下载配置文件，放入手机目录（data/data/com.termux/files/home）


6、 打开超级终端依次输入命令：pkg install python
python sunny.py
打开浏览器访问查看是否成功
