# 缘起

今天看了电影《热辣滚烫》

真的很励志。

电影讲述的是乐莹（贾玲）毕业后在家宅十年后，遭受种种打击后励志打完一场拳击比赛，最终通过一年的努力完成蜕变的故事。

# 拼搏

看到这里我也在拷问自己：😭

我赢过吗？

哪怕一次！

# 绝望

乐莹在

男友劈腿闺蜜

妹妹想要房产

朋友欺骗利用

拳击教练事后概不负责

四重打击下彻底绝望，跳楼未果后，选择赢一次！💖

# 汗水

开启了自己热辣滚烫的奋斗！👟

# 不弃

经历春夏秋冬🍀🍁🍂🍃，四季变迁。这一天终于到来！

乐莹迈向赛场，内心早已坚定，当初的乐莹不复存在。【泪目😭过往的经历，四季的汗水浮现】

面对职业拳击手，乐莹应声倒地。

但！

她并没输。

因为只要完整打完拳击比赛，就已经赢了！【泪目😭】

# 吸引

打完比赛回家时，曾抛弃自己的健身教练又来约乐莹吃饭。

乐莹内心早已充盈。

不再是当初满眼是你的女孩。

# 未来

可是

未来还有很多机会，对吗？

# 撒花

一年的历程历历在目，太佩服贾玲了。我也可以吗？

+++

# YOLO

You Only Live Once!

你只活一次！

+++

结束！大家可以返回文章了。

如果看到了这里：那么我想和你交个朋友！

未来规划：🎀

真的哭了。

觉得自己又何尝不是之前的乐莹呢？

喜欢的事没勇气去做

喜欢的人不喜欢自己

浑浑噩噩一年又一年

下定的决心破了又破

长相不出众邋里邋遢

学习没学好普普通通

玩也没玩明白不开朗

程序员盯电脑要捉瞎

久坐身体也每况日下

今天是农历三月十三，真的刚好 24 年整。🎥

回想过去 24 年的经历，我又何尝不是乐莹呢？

唯一坚持下来的或许就是

活着！

可我真的甘心这样吗？

渴望的爱情会到来吗？

升职加薪会轮到我吗？

😭

+++

看了《热辣滚烫》

我想，一切都有了答案！🏮

放弃那个很喜欢的人！

丢掉那些坏习惯！

不再管他升职加薪！

内心坚定不再迷茫！

我要做的只有一件事：

遇见更好的自己！🎖

不再去想如何讨好她

只剩下提升自己

不再去自我怀疑

只是去做

不再精神内耗

都是 NPC

乐莹可以一年蜕变！

我

也可以！

因为我们，只活一次！

+++

写下这篇日记

是为了当自己坚持不下去时，想要放弃时，来这里看看。

警醒自己，YOLO！

+++

规划：【不用太苛刻自己】

  • 英语：每天早上刷墨墨背单词
  • 舞蹈：每周三 & 周日去舞蹈室学 Hiphop
  • 健身：每天晚上 7 点公园跑步 & 21 点 keep
  • 护肤：早晚坚持护肤补水防晒
  • 穿搭：跟着舞蹈老师学穿搭
  • 饮食：戒辣，戒糖
  • 攒钱：每月攒下 3000
  • 文章：筹划一个 AutoSar 的专栏
  • 视频：哈哈哈等学会跳舞再说吧
  • 社交：每周都认识一些新朋友

就这样了！【背英语断了一天就断了，没什么大不了的，明天继续背】

放平心态才能长久！

+++

加油！柴林峰，You Only Live Once！去活出你想要的生活吧！

相信，当我变成更好的自己后，终将邂逅属于自己的爱情！
