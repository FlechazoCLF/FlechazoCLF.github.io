# arduino IDE 配置

第一步：在电脑上下载 Arduino ide。并安装好 ide。我用的是 1.6.5 版本的 IDE。

第二步：点击 ide 中的 “文件”—” 首选项 “。

第三步：填写网址：http://arduino.esp8266.com/stable/package_esp8266com_index.json

第四步：点击 ide 中的 “工具”—“板”—“boards manager…”

第五步：出现界面，随后在此界面中，搜索 “esp8266”

第六步：在右下角选择最新版本，并点击”install” 进行安装完成。

第七步：点击” 工具 “—” 板 “—”Generic ESP8266 Module
