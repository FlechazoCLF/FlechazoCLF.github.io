# wake up

/weik up/ 醒来

# get up

/get up/ 起床

# wash my face

/wash mai feis/ 洗脸

# go to the bathroom

/gou tu the ‘bae,thrum/ 上厕所

# dry my face

/drai mai feis/ 擦干脸

# brush my teeth

/brush mai tith/ 刷牙

# take a shower

/teik a ’shauer/ 洗澡

# shave

/sheiv/ 刮胡子

# get dressed

/get drest/ 穿衣服

# comb my hair

/koum my hair/ 梳头

# put on makeup

/put an ‘mei,kup/ 化妆

# eat breakfast

/it ’breakfast/ 吃早餐

# have a cup of tea

/haev a kup uv ti/ 喝杯茶

# check my phone

/tchek mai foun/ 看手机

# read the paper

/rid the ‘paper/ 读报纸

# listen to the radio

/’lisen tu the ‘reidi,ou/ 听收音机

# go to school

/gou tu skul/ 上学

# go to work

/gou tu work/ 上班

# get off work

/get of work/ 下班

# take the bus

/teik the bus/ 搭巴士（坐公交）

# take the subway

/teik the ’su,bwei/ 坐地铁

# have lunch

/haev luntch/ 吃午饭

# exercise

/‘ekser,saiz/ 锻炼

# work out

/werk aut/ 健身

# go home

/gou noum/ 回家

# do homework

/du ’houm,work/ 做作业

# make dinner

/meik ‘diner/ 做晚饭

# have dinner

/haev diner/ 吃晚饭

# take a walk

/teik a wok/ 散步

# do housework

/du ’hau,swerk/ 做家务

# take a bath

/teik a baeth/ 洗个澡

# watch TV

/watch ‘ti’vi/ 看电视

# take a nap

/teik a naep/ 打盹

# go to bed

/gou tu bed/ 上床

# sleep

/slip/ 睡觉
