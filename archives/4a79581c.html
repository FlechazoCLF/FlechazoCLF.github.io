# 小柴冲刺软考中级嵌入式系统设计师系列一、计算机系统基础知识（1）嵌入式计算机系统概述

根据 IEEE（国际电气电子工程师协会）的定义，嵌入式系统是 " 控制、监视或者辅助设备、机器和车间运行的装备 "。

# 一、计算机硬件

基本的计算机硬件系统由运算器、控制器、存储器、输入设备和输出设备 5 大部件组成。

运算器和控制器及其相关部件被集成在一起，统一为中央处理单元（Central Processing Unit,CPU）。

部件作用
运算器对数据进行加工处理 <br /> 完成算数和逻辑运算
控制器从主存中取出指令并进行分析
存储器内部存储：速度快、容量小 <br /> 外部存储：速度慢、容量大 < br /> 寄存器：临时存少量数据、运算结果、正在执行的指令
输入 / 输出设备与外界交换信息

# 二、计算机软件

通常可分为系统软件、中间件和应用软件等。

软件类型作用
系统软件管理系统的硬件和软件资源
中间件独立的系统软件或服务程序 <br /> 常用来管理计算资源和网络通信 < br /> 提供通信处理、数据存储、事务处理、Web 服务、安全、跨平台等服务
应用软件用于解决应用领域的具体问题
