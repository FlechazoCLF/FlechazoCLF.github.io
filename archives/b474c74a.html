⛄用旧手机做服务器搭建网站☃

发表于 2022-08-08
  • flechazo
首页 技术教程🎃 ksweb

对于旧手机，我们大可拿去回收网站回收，或者咸鱼转卖。

但我们还可以选择将旧手机改造成一台 web 服务器（再加上 FRP 内网穿透，安排的妥妥的）。

当然，对硬件还是有一定要求的，手机的硬件配置基础要求是：CPU 主频不低于 600MHZ，内存大于 256MB，闪存大于 512MB。这个配置要求以现在的智能手机标准来看是可以完全满足的。

搭建服务器我们需要用到软件

1.KSWEB

2. 樱花 FRP

打开后软件是这样的，我这个是已经做好的。我们要先配置数据库。打开工具页面，选择 phpMyAdmin

选择服务器：Lighttpd 点击确认，记住这里分配的端口为 8000

(如果你是 Nginx 作为 WEB 服务程序可以勾选 Nginx)

然后就是主机配置

端口可以改成 80，这样直接输入 ip 就能访问了。不过还是建议使用默认端口。

然后登录数据，用户名是 root，密码留空

在数据库中创建一个数据库

基本的配置已经完成了。

下面开始配置程序搭建网站

以 wordpress 程序为例，先下载程序压缩包。这个个可以百度 wordpress 官网进行下载。然后把程序解压到目录 htdocs 里面。打开浏览器访问 ip:127.0.01 或者 http://localhost:8080（端口具体根据自己的设置）

它会提示安装

配置数据库时

数据库名一定要填写刚刚创建的数据库名字，因为没有密码所以密码留空就行了。然后确认下一步

这些东西自己设置填写，然后就安装成功了。

用设置的账号密码登录后台即可。

这个只能在内网访问，想在互联网中访问需要内网穿透。有兴趣的可以去百度一下。

若有错误，请指出。谢谢
